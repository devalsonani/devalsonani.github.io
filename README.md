# Deval Sonani — Software Engineer Portfolio

Welcome to the source code for my professional portfolio!

This is a high-performance, fully responsive, glassmorphism-themed personal website built with Vanilla HTML, CSS, and JavaScript. It serves as a digital resume and showcase of my work in full-stack development, distributed systems, and AI-powered solutions.

## 🚀 Live Demo

**[View the Live Portfolio](https://devil3101.github.io/)**

## ✨ Features

- **Modern UI/UX**: Premium dark theme featuring glassmorphism and subtle neon accents.
- **Interactive Animations**: Custom particle engine background, dynamic typing effects, scroll reveals, and smooth animated skill progress bars.
- **Responsive Layout**: Fluid design that works beautifully on desktops, tablets, and mobile devices.
- **Zero Dependencies**: Built entirely with Vanilla JS and CSS for maximum performance and lightning-fast load times.
- **Automated Deployments**: Integrated with GitHub Actions for seamless continuous deployment to GitHub Pages.

## 🛠 Tech Stack

- **HTML5**: Semantic and accessible structure.
- **CSS3**: Custom properties (variables), Flexbox/Grid layouts, and CSS animations.
- **JavaScript (ES5/ES6)**: IntersectionObserver API for scroll interactions, Canvas API for the particle system.
